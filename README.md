# synchrsub

ASS editor for web

**[Use online (Github Page)](https://op200.github.io/synchrsub/)**


### Libraries used

#### Web framework

* [Vue 3](https://vuejs.org)
* [Vue Router](https://router.vuejs.org)
* [Pinia](https://pinia.vuejs.org)

#### Web ASS renderer

* [ASS.js](https://github.com/weizhenye/ASS)
